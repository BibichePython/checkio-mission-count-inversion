In computer science and discrete mathematics, an inversion is a pair of places of a sequence where the elements on these places are out of their natural order. So if we use ascending order for numbers, then a inversion is when bigger number appear before lower number in a sequence.

Let's look at example. sequence = (1, 2, 5, 3, 4, 7, 6) and we can see here three inversion:
- 5 and 3; - 5 and 4; - 7 and 6.

You are given a sequence of unique numbers and you should count inversion in this sequence.

Input: A sequence as a tuple of integers.

Output: The inversion number as an integer.

Example: 

count_inversion((1, 2, 5, 3, 4, 7, 6)) == 3
count_inversion((0, 1, 2, 3)) == 0

How it is used: In this mission you will work with nested loops, of course if you don't use advanced algorithms.

Precondition: 2 < len(sequence) < 200
len(sequence) == len(set(sequence))
all(-100 < x < 100 for x in sequence)
